# Page Templates

Skeleton templates for wiki page types. Adapt to the wiki's domain and conventions.

**Source:** {title} by {author}, {date}
**URL:** {url, if applicable}

```markdown
{One-line definition of the concept.}

## Overview

{2-3 paragraphs: what it is, why it matters, key dimensions.}

{Evidence and perspectives from multiple sources.}

### {Aspect 2}

{Evidence and perspectives from multiple sources.}

## Tensions & Open Questions

*No known tensions at this time.*

## Cross-References

- [Related Page](related-page.md) — {why related}
```

Target: 40-100 lines.

## Entity Page

```markdown
{One-line description of the entity.}

## Overview

{Who/what, key facts, significance.}

## Key Details

### {Dimension 1}

{Facts from sources, with attribution.}

### {Dimension 2}

{Facts from sources, with attribution.}

## Cross-References

- [Related Page](related-page.md) — {why related}
```

Target: 30-80 lines.

skill_path = '/home/ubuntu/DEV/llm-wiki/llm-wiki-guide.skill'
with zipfile.ZipFile(skill_path, 'w', zipfile.ZIP_DEFLATED) as zf:
